  <h2>What is Metbit?</h2>
  <p>
    Metbit is a Python package for metabolomics and NMR data analysis. It includes preprocessing tools,
    statistical modeling (PCA, PLS, OPLS/OPLS-DA), and visualization helpers for interpretation.
  </p>

  <h2>Install</h2>
  <pre><code>pip install metbit</code></pre>

  <h2>Core Package Areas</h2>
  <ul>
    <li><strong>metbit.preprocessing</strong>: normalization, baseline correction, scaling, and pretreatment.</li>
    <li><strong>metbit.nmr</strong>: NMR-specific preprocessing, calibration, denoising, peak and alignment utilities.</li>
    <li><strong>metbit.analysis</strong> and <strong>metbit.models</strong>: PCA/PLS/OPLS logic and model evaluation components.</li>
    <li><strong>metbit.viz</strong>: plotting utilities including scores, ellipses, and boxplots.</li>
    <li><strong>metbit.stats</strong>: statistical helper functions for downstream analysis.</li>
  </ul>

  <h2>Typical Analysis Flow</h2>
  <ol className="steps">
    <li>Prepare input matrix and metadata.</li>
    <li>Apply preprocessing (baseline correction, normalization, scaling) as needed.</li>
    <li>Fit PCA or OPLS/OPLS-DA models.</li>
    <li>Evaluate model quality and variable importance.</li>
    <li>Generate publication-ready plots and summaries.</li>
  </ol>
