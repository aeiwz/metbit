# nmr_preprocess

> Category: Data Processing

## Classes

### nmr_preprocessing 

A class for preprocessing NMR data.
This class handles the following preprocessing steps:
1. Reading FID files
2. Zero-filling
3. Fourier Transform
4. Phasing
5. Baseline correction
6. Calibration
7. Data storage in a pandas DataFrame
8. Data visualization
9. Data export
