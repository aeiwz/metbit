## Functions

### `nipals(x, y, tol, max_iter, dot)`

Non-linear Iterative Partial Least Squares

#### Parameters

- `x`: np.ndarray Variable matrix with size n by p, where n number of samples, p number of variables.
- `y`: np.ndarray Dependent variable with size n by 1.
- `tol`: float Tolerance for the convergence.
- `max_iter`: int Maximal number of iterations.

#### Returns

- `w`: np.ndarray Weights with size p by 1.
- `u`: np.ndarray Y-scores with size n by 1.
- `c`: float Y-weight
- `t`: np.ndarray Scores with size n by 1 References ----------
